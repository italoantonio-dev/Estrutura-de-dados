# Aula 04 — Refatoração e Métodos | Class 04 — Refactoring and Methods

---

## 🇧🇷 Português

### Sobre a Aula

Consolidação dos conceitos das aulas anteriores, refatorando os exercícios em **métodos reutilizáveis** e expandindo a compreensão de **organização de código**.

### Objetivos

- Refatorar exercícios anteriores em métodos bem estruturados
- Compreender a reutilização de código
- Aplicar boas práticas de nomeação e organização

### Exercícios

| Arquivo | Descrição |
|---|---|
| `CaixaEletronico.java` | Refatoração do caixa eletrônico com método separado para cálculo de troco |
| `ContadordeNotas.java` | Contagem e análise de notas escolares |
| `Senha.java` | Refatoração da validação de senha em método dedicado |
| `Tempo.java` | Refatoração do conversor de tempo com métodos separados |
| `VerificarUnico.java` | Refatoração da verificação de elementos únicos em método |

### Como Compilar e Executar

```bash
cd "Aula04"
javac src/CaixaEletronico.java
java -cp src CaixaEletronico
```

## 🇺🇸 English

### About This Class

Consolidation of concepts from previous classes, refactoring exercises into **reusable methods** and expanding understanding of **code organization**.

### Objectives

- Refactor previous exercises into well-structured methods
- Understand code reusability
- Apply best practices in naming and code organization

### Exercises

| File | Description |
|---|---|
| `CaixaEletronico.java` | Refactored ATM with dedicated method for change calculation |
| `ContadordeNotas.java` | School grades counting and analysis |
| `Senha.java` | Refactored password validation in a dedicated method |
| `Tempo.java` | Refactored time converter with separate methods |
| `VerificarUnico.java` | Refactored unique element verification in a method |

### How to Compile and Run

```bash
cd "Aula04"
javac src/CaixaEletronico.java
java -cp src CaixaEletronico
```
